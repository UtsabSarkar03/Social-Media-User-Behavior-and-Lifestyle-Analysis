# Social Media User Behavior and Lifestyle Analysis

A comprehensive exploratory data analysis (EDA) project examining the relationship between social media usage patterns and user lifestyle metrics, with a focus on wellbeing indicators and behavioral trends.

## üìä Project Overview

This project analyzes a dataset of 100,000 social media users to uncover insights about digital consumption patterns and their correlation with various lifestyle factors. The analysis explores multiple dimensions including demographics, usage behavior, content preferences, and self-reported wellbeing metrics.

## üìä [Data Link]([url](https://www.kaggle.com/datasets/rockyt07/social-media-user-analysis))

## üéØ Objectives

- Analyze social media usage patterns across different demographic segments
- Investigate the relationship between digital consumption and user wellbeing
- Identify trends in content preferences and engagement metrics
- Examine correlations between various behavioral and lifestyle factors
- Provide data-driven insights for understanding modern social media behavior

## üìÅ Dataset

The dataset contains 100,000 user records with the following key features:

**Demographic Information:**
- Age
- Gender
- Location (Country, State, City)

**Usage Metrics:**
- Daily screen time
- Posts per day
- Likes received per day
- Comments received per day
- Messages sent per day
- Reels watched per day

**Behavioral Patterns:**
- Video watch time per day
- Time spent on platform per day
- Notifications received per day

**Lifestyle & Wellbeing:**
- Self-reported happiness score
- Productivity loss (hours/day)
- Sleep hours per day
- Online shopping frequency
- Dominant emotion

**Platform Preferences:**
- Top interests
- Connection type (WiFi/Mobile Data)
- Addiction level
- Gender of most-liked content creators

## üîß Technologies Used

- **Matplotlib** - Data visualization

## üîç Key Findings

The analysis reveals several important insights about social media user behavior:

1. **Usage Intensity**: Varied patterns of engagement across different user segments
2. **Wellbeing Impact**: Notable correlations between usage metrics and self-reported happiness
3. **Demographic Variations**: Distinct behavioral patterns across age groups and genders
4. **Content Preferences**: Clear trends in preferred content types and consumption habits
5. **Productivity Effects**: Measurable relationship between social media usage and productivity metrics
